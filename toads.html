Toads with Guns:

Toads with Guns is a stylized tower defense game developed in Unity that blends classic strategy gameplay with a unique swamp-noir aesthetic. Players defend their territory as armed toads against waves of mutant insects, strategically placing towers, managing resources, and adapting to increasingly complex enemy behaviors. The project emphasizes modular game architecture, with core systems such as enemy spawning, pathfinding, combat targeting, and game state management built using a combination of C# classes and Unity components. Designed as a team-based software engineering project, Toads with Guns highlights both technical implementation and collaborative development, showcasing skills in game systems design, object-oriented programming, and real-time interactive gameplay.
